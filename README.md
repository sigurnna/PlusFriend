# 소개
카카오톡 플러스친구 - 강릉원주대학교 학식봇의 API 서버입니다.
Firebase functions를 이용하여 서버리스 형태로 개발되었습니다.

# 사용한 프레임워크
- Firebase functions
- Express.js

# 구조
/functions/index.js 에 카카오톡 플러스친구 스마트채팅 대응을 위한 api 가 작성되어있습니다.
